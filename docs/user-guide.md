# Special-live 用户指南

## 1. 概述

Special-live 是一个基于 SRS v6 的浏览器直播平台。它提供了一个简洁直观的界面，让你可以观看直播流，管理员可以管理房间、推流密钥和播放源。

主要功能：

- 基于 mpegts.js 的 FLV 播放（桌面端）
- iOS Safari 自动切换到 HLS 播放
- 通过 Authentik OAuth2 登录
- 管理员面板管理站点设置、房间和密钥
- 支持公开/私密房间
- 手动模式下可配置多画质播放源

---

## 2. 登录

平台使用 Authentik 进行统一身份认证。

**登录步骤：**

1. 在页面右上角点击 **Sign In** 按钮。
2. 系统会跳转到 Authentik 登录页面。
3. 使用你在 Authentik 中的账号完成登录。
4. 登录成功后，页面会自动跳转回首页。

**关于管理员：**

- 平台中第一个登录的用户会被**自动提升为管理员**。
- 之后登录的用户默认为普通用户。
- 只有管理员能看到和使用管理面板。

---

## 3. 观看直播

### 首页

- 首页以卡片网格的形式展示所有**公开房间**。
- 卡片只显示房间名称，不显示直播状态或观看人数。
- 点击任意卡片，进入该房间的播放页面。
- 如果没有可用房间，页面会显示 "No streams available"。

### 播放器

- 进入房间后，页面会切换到**全屏播放器**（ArtPlayer）。
- 播放器会自动播放流，无其他界面干扰。
- 桌面端（Chrome / Firefox / Edge）使用 **mpegts.js** 播放 FLV 流。
- iOS Safari 不支持 FLV，系统会自动切换到 **HLS** 播放。

### 私密房间

- 私密房间不会出现在首页列表中。
- 你可以通过直接访问链接来观看私密房间：
  - 在 URL 后附加访问密钥：`/live/[房间slug]?key=你的密钥`
  - 或者先在平台上登录，已登录用户可以观看私密房间。

---

## 4. 管理面板（管理员）

登录管理员账号后，侧边栏会出现 **Admin** 入口。点击进入管理面板。

管理面板包含以下管理页面：

### 站点设置 (Site Config)

- **站点标题 (Site Title)**：修改网站显示的名称。
- **Favicon URL**：修改浏览器标签页上的网站图标。输入 URL 后可以预览效果。
- 修改后点击 **Save** 保存。

### 房间管理 (Rooms)

房间列表页展示所有房间，包含以下信息：

| 列 | 说明 |
|---|---|
| Name | 房间名称，点击进入房间详情 |
| Slug | 房间的 URL 标识，自动生成 |
| Visibility | 公开/私密（绿色/红色标签） |
| Stream Key | 推流流密钥（仅显示前 8 位） |
| Manual Mode | 开关手动播放源模式 |
| Created At | 创建时间 |
| Actions | 编辑/删除操作 |

**创建房间：**

1. 点击 **Create Room** 按钮。
2. 输入房间名称，Visibility 选择公开或私密。
3. 点击 **Create** 完成创建。

**删除房间：**

- 点击房间右侧的删除按钮，确认后删除。
- 删除房间会同时删除该房间的所有推流密钥和播放源。

**房间详情：**

点击房间名称进入详情页，包含三个标签页：

1. **General**：修改房间名称和公开/私密属性，点击 Save Changes 保存。
2. **Keys**：管理该房间的推流密钥。
3. **Play Sources**：管理手动模式下的播放源。

### 密钥管理 (Keys)

每个房间可以生成多个推流密钥。

**生成新密钥：**

1. 在房间详情页进入 **Keys** 标签页。
2. 点击 **Generate New Key**。
3. 输入密钥标识（例如 "OBS Studio"），点击 Generate Key。
4. **密钥会一次性明文显示在屏幕上——请立即复制并保存。**
5. 关闭弹窗后，密钥将不再显示。

**管理密钥：**

- **启用/禁用**：使用 Switch 开关切换密钥的 Active / Inactive 状态。禁用的密钥无法用于推流。
- **删除密钥**：点击删除按钮。正在使用该密钥的推流会立刻断开。

### 播放源管理 (Play Sources)

当房间开启了 **Manual Mode**（手动模式）时，可以自定义播放源。

**添加播放源：**

1. 在房间详情页进入 **Play Sources** 标签页。
2. 点击 **Add Source**。
3. 输入来源名称（例如 "Primary CDN"）。
4. 至少添加一个画质配置：**标签**（例如 1080p、720p）和 **播放 URL**。
5. 一个源可以添加多个画质。
6. 点击 **Create Source** 保存。

**注意：** 如果 Manual Mode 未开启，播放源不会生效，播放器会使用默认的 FLV 地址。开启 Manual Mode 后，播放源会覆盖默认播放行为。

## 5. 推流 (OBS)

### 设置步骤

1. 打开 OBS Studio。
2. 进入 **设置 → 推流**。
3. 按以下方式配置：

| 设置项 | 值 |
|---|---|
| 服务 | 自定义 |
| 服务器 | `rtmp://live-push.example.com/live/` |
| 串流密钥 | `房间slug?token=你的推流密钥` |

4. 点击 **确定** 保存设置。
5. 在 OBS 中点击 **开始推流**。

### 推流密钥格式说明

推流密钥需要包含**房间标识**和**推流密钥**两部分，用 `?token=` 连接：

```
房间slug?token=你的推流密钥
```

例如，如果你的房间 slug 是 `my-game-stream`，推流密钥是 `sk_abc123xyz`，则串流密钥填写：

```
my-game-stream?token=sk_abc123xyz
```

### 注意事项

- 推流密钥在生成时**只显示一次**，请妥善保存。
- 如果忘记密钥，在管理面板中生成新密钥即可。旧密钥可以禁用或删除。
- 一个房间可以有多个密钥同时处于启用状态。

## 6. 常见问题

**iOS 上看不了直播？**

iOS Safari 不支持 FLV 格式。平台会自动检测 iOS 设备并切换到 HLS 播放。如果仍然无法播放，请确认 SRS 服务端已启用 HLS 输出。

**忘记推流密钥了怎么办？**

在管理面板中进入对应的房间详情页，在 Keys 标签页中生成一个新的推流密钥。如果旧密钥不再需要，可以将其禁用或删除。

**如何让别人观看私密房间？**

有两种方式：

1. 分享私密房间的访问密钥。对方可以通过 URL 附带 `?key=密钥` 来观看。
2. 让对方在平台注册 Authentik 账号，登录后可直接通过链接观看。

**如何在管理面板修改站点标题？**

在侧边栏的 Admin 菜单中点击 **Site Config**，修改 Site Title 后保存即可。

**Manual Mode 是什么？**

Manual Mode（手动模式）允许管理员为房间配置自定义的播放源和画质。开启后，你可以添加一个或多个播放源（如 CDN 地址），每个播放源可以包含多个画质（如 1080p、720p）。播放器会根据配置切换画质。手动模式关闭时，播放器使用默认的 FLV 地址播放。

**推流失败怎么办？**

- 检查 OBS 推流设置是否正确（服务器地址和串流密钥）。
- 确认推流密钥是否已启用（Active）。
- 确认房间 slug 填写正确。
- 检查网络连接能否到达推流服务器。
